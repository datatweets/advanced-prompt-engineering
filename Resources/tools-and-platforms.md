# Tools And Platforms

This course is tool-neutral. Learners can use any approved AI platform available to them.

## Tool Categories

| Category | Use |
| --- | --- |
| Chat assistants | Drafting, summarizing, brainstorming, rewriting |
| Document tools | Working with long files, reports, and notes |
| Image tools | Creating visual concepts, mood boards, and campaign assets |
| Search-enabled tools | Current research and source discovery |
| Automation tools | Repeating approved workflows |
| Collaboration tools | Sharing prompts, outputs, and review comments |

## Selection Criteria

Before choosing a tool, ask:

- Is it approved for the data I will use?
- Can it access current information if I need current facts?
- Does it support file upload if I need document analysis?
- Can I control privacy and retention settings?
- Is the output easy to export and review?
- Does the organization allow this tool for the task?

## Responsible Use Note

Tool features and policies change. Check current official documentation before using a tool with sensitive data, regulated work, or public claims.
