# Glossary

| Term | Plain English meaning |
| --- | --- |
| AI | Software that performs tasks that normally require human thinking |
| Generative AI | AI that creates new content such as text, images, audio, or code |
| Prompt | The instruction, question, or task given to AI |
| Prompt engineering | The practice of designing prompts for better outputs |
| Context | Background information that helps AI respond well |
| Token | A small unit of text processed by a model |
| Hallucination | A confident answer that is wrong or unsupported |
| Grounding | Connecting AI output to provided facts or sources |
| Zero-shot prompt | A prompt without examples |
| Few-shot prompt | A prompt that includes examples |
| Persona | A practical summary of an audience segment |
| Content strategy | A plan for what to publish, for whom, where, and why |
| Storyboard | A scene-by-scene plan for visual or multimedia content |
| Governance | Rules and processes for responsible use |
| Human review | A person checking AI output before use |
