## Example: Team Guideline Prompt

```text
Draft AI usage guidelines for a small marketing team.

Approved uses:
- Drafting outlines
- Summarizing non-confidential notes
- Creating content variations
- Reviewing tone and clarity

Restricted uses:
- Customer personal data
- Legal advice
- Financial projections
- Publishing without review

Return a one-page policy in plain English.
```
