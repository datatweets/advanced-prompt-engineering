# Module 3: Content Creation

![Creation](https://img.shields.io/badge/focus-content%20creation-f97316)
![Output](https://img.shields.io/badge/output-drafts%20and%20storyboards-16a34a)

This module shows how to use prompt engineering to create business and learning content. It covers blog posts, social media content, email drafts, video concepts, and digital storyboards.

The goal is not to let AI write everything without review. The goal is to use AI as a drafting, editing, planning, and variation tool.

## Duration

Total module time: 150 minutes + 60-minute bonus lesson.

| Lesson | Time | Teaching focus |
| --- | ---: | --- |
| [01-writing-blog-posts.md](01-writing-blog-posts.md) | 50 min | Five-step article workflow, content brief, section-by-section drafting, and quality review |
| [02-social-media-content.md](02-social-media-content.md) | 50 min | Bait-hook-reward-payload framework, planning before writing, and five opening lines test |
| [03-digital-storyboards.md](03-digital-storyboards.md) | 50 min | Narrative structure, scene planning, weak-vs-improved comparisons, and storyboard review |
| [04-image-generation-tools.md](04-image-generation-tools.md) *(bonus)* | 60 min | DALL-E, Midjourney, and Stable Diffusion — prompt anatomy, style control, negative prompts, and responsible use |

## Learning Outcomes

After this module, learners should be able to:

1. Convert a content goal into a structured multi-step prompt workflow.
2. Control audience, tone, format, and length at each stage of drafting.
3. Apply the bait-hook-reward-payload framework to social content.
4. Plan and write a five-scene digital storyboard.
5. Revise AI output using an explicit quality criteria table.
6. *(Bonus)* Write image prompts for DALL-E, Midjourney, and Stable Diffusion, and choose the right tool for each content job.

## Module Scenario

A training provider wants to launch a short campaign for a new professional course. The team needs a blog article, social posts, an email, and a storyboard for a short video.

## Content Rule

Treat the first AI output as a draft. Review for accuracy, audience fit, originality, tone, and business risk before publishing.
