# Module 3 Examples

## Example: Blog Planning Prompt

```text
Act as a business writing assistant.

Create a detailed outline for a blog post.

Audience:
Managers who want to use AI for weekly reports.

Goal:
Teach a simple workflow they can test this week.

Return:
1. Title options
2. Recommended structure
3. Key points for each section
4. Practical example
5. Review checklist

Rules:
- No unsupported statistics.
- No hype.
- Use plain English.
```

## Example: Repurposing Prompt

```text
Turn the blog outline below into a content package.

Return:
1. Professional network post
2. Email teaser
3. Internal announcement
4. Short video storyboard
5. Three audience-specific angles

Blog outline:
[paste outline]
```
