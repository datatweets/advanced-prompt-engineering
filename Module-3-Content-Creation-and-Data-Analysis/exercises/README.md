# Module 3 Exercises

## Exercise 1: Blog Draft And Review

Write a prompt for a 600-word blog post. Include:

- Audience.
- Topic.
- Goal.
- Tone.
- Required sections.
- Claims to avoid.

Then ask AI to review the draft for vague claims and missing examples.

## Exercise 2: Channel Adaptation

Take one blog idea and turn it into:

- One professional network post.
- One email teaser.
- One internal announcement.
- Five short hook options.

## Exercise 3: Storyboard

Create a storyboard for a 60-second explainer video or a five-slide lesson.

Required columns:

| Scene | Goal | Visual | On-screen text | Narration | Notes |
