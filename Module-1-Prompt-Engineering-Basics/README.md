# Module 1: Prompt Engineering Basics

![Concepts](https://img.shields.io/badge/focus-foundations-2563eb)
![Practice](https://img.shields.io/badge/output-first%20usable%20prompts-16a34a)

This module explains what AI is, how generative AI responds to prompts, and how to write instructions that produce useful results. It is built from the notes on tokens, ChatGPT capabilities, standard text practices, and the five principles of prompting.

## Duration

Total module time: 150 minutes.

| Lesson | Time | Teaching focus |
| --- | ---: | --- |
| [01-what-is-ai.md](01-what-is-ai.md) | 50 min | AI, tokens, context, capabilities, how AI systems are built, and two types of AI work |
| [02-prompt-engineering-principles.md](02-prompt-engineering-principles.md) | 50 min | Five principles, prompt anatomy, nine sharpening techniques, and chain-of-thought |
| [03-writing-your-first-prompt.md](03-writing-your-first-prompt.md) | 50 min | Build and test prompts, bad-better-best pattern, and the ask-me-first technique |

## Learning Outcomes

After this module, learners should be able to:

1. Explain generative AI in plain English and describe how modern AI systems are structured.
2. Identify all eight elements of a well-built prompt.
3. Use role, task, context, constraints, format, and a success brief.
4. Apply nine practical techniques to sharpen any prompt.
5. Improve prompts using the bad-better-best pattern.
6. Use the ask-me-first technique to clarify before writing.
7. Review AI output against stated quality criteria.

## Module Scenario

A small company wants to use AI to answer common customer questions, draft social posts, summarize meeting notes, and prepare internal reports. The team has mixed experience: some learners are managers, some are employees, and some are students. The goal is to create prompts that are clear enough for work but simple enough for beginners.

## Recommended Flow

1. Read each lesson in order.
2. Complete one exercise after each lesson.
3. Save the best prompt from each exercise in a personal prompt library.
4. Compare outputs before and after prompt improvement.
