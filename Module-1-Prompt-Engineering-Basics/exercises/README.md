# Module 1 Exercises

## Exercise 1: Explain AI To A Specific Audience

Choose one audience:

- Business owner.
- Department manager.
- Corporate employee.
- Student.

Write a prompt that asks AI to explain generative AI to that audience.

Your prompt must include:

- Audience.
- Goal.
- Format.
- One realistic example.
- One limitation or risk.

## Exercise 2: Improve A Weak Prompt

Improve this prompt:

```text
Write something about my business.
```

Add:

- Business context.
- Audience.
- Channel.
- Tone.
- Length.
- Success criteria.

## Exercise 3: Prompt Review

Run your improved prompt in an AI tool. Then ask:

```text
Review your previous answer for unclear claims, missing context, and practical usefulness. Suggest improvements in a table.
```

Revise the prompt one more time.

## Exercise 4: Zero-Shot, One-Shot, Few-Shot

Choose one task:

- Classify customer feedback.
- Rewrite a support message.
- Summarize meeting notes.
- Create content ideas.

Write three prompt versions:

1. Zero-shot: no examples.
2. One-shot: one example.
3. Few-shot: three examples.

Run all three versions. Compare:

- Which output was most consistent?
- Which output was most creative?
- Which output followed the format best?
- Which version would you use for repeated business work?

## Exercise 5: Step-By-Step Reasoning

Write a prompt for a task that needs judgment, such as choosing a content topic, prioritizing customer complaints, or planning a campaign.

Your prompt must ask AI to return:

- Final answer.
- Key assumptions.
- Missing information.
- Human verification checklist.

Avoid asking for a long private reasoning trail. Ask for concise, reviewable reasoning.
