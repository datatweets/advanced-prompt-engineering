# Module 2 Examples

## Example: Audience Research Prompt

```text
Act as a market research assistant.

Analyze these customer comments.

Return a table with:
- Theme
- Supporting comments
- Possible audience segment
- Content opportunity
- Confidence level

Rules:
- Use only the provided comments.
- Label assumptions clearly.

Comments:
[paste comments]
```

## Example: Persona Review Prompt

```text
Review these persona cards.

Find:
1. Unsupported assumptions
2. Missing evidence
3. Stereotypes or unfair generalizations
4. Better content opportunities

Return a table with issue, why it matters, and recommended fix.
```
